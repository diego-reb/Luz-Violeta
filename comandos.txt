GIT 


1. Comando para enlistar archivos de la carpeta 
	ls

2. Comando para buscar carpeta
	cd nombre de tu carpeta


3. Comando para ver el estatus de los archivos 
	git status

4. Comando para agregar todos los archivos 
	git add .

5. Comando para agregar 1 archivo o ciertos archivos 
	git add nombre del archivo 

6. Comando para hacer un commit 
	git commit -m "Comentario de lo que realizaste"

7. Comando para subir los archivos 
	git push 

8. Comando para descargar los archivos 
	git pull 

//Los comando estan acomodados en el orden que se deben ejecutar
